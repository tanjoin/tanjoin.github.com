# PASMO

## 払戻し金額

```
SF残額 + デポジット 500円
```

## 払戻し場所

PASMO・Suicaエリアの駅やバス営業所等

## 謎の用語

### SF（ストアードフェア）

PASMOにチャージ（入金）された、運賃に使用できる金額のことです。
