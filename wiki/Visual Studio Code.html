インストール

Download Visual Studio Code - Mac OS X, Linux, Windows

Command Palette

Ctrl + Shift + P

Ctrl + P はファイル検索

言語モード／改行コード／エンコーディング

右下で設定できる

code コマンド

code .
で簡単に開けるらしいが Github の MINGW32 では使えなかった

何かしらの設定をすればできるだろう

IntelliSense

Ctrl + Space

Multi-cursor

Ctrl + D で複数行同時編集モード

テーマ変更

1. Ctrl + Shift + P

2. Type theme

3. Select theme

ユーザ定義スニペット

1. Ctrl + Shift + P

2. Type snippets

3. Select Snippets

C:\Users\[username]\AppData\Roaming\Code\User\snippets

に保存される

"p tag": {

  "prefix": "p",

  "body": [

    "<p>$1</p>",

    "$2"

  ],

  "description": "<p></p>"

}

$1 はショートカット実行後のカーソルの位置っぽい

#2 は不明（未確認）

Workspace Settings

2. Type settings
